## Image Recommendations

- **Format**: JPG or PNG
- **Cat Photos**: 800x600px or similar aspect ratio
- **Hero Image**: 1920x600px (wide landscape)
- **Team Photos**: 300x300px (square)
- **File Size**: Optimize images to under 500KB each for faster loading

## Free Stock Photo Resources

You can find free cat photos at:
- Unsplash.com
- Pexels.com
- Pixabay.com

Search for: "cat portrait", "tabby cat", "siamese cat", etc.

## Note

The website will still function without images - it will show broken image icons or use the placeholder fallback. However, adding real images will greatly enhance the user experience!
